# Artifacts Documentation

This directory contains references to the artifacts and datasets utilized by LexFusionNet.

## Dataset References
- **Primary Dataset**: Supreme Court of India Judgments (1950-2024)
- **Source**: [Kaggle](https://www.kaggle.com/datasets/adarshsingh0903/legal-dataset-sc-judgments-india-19502024)

## Schema Explanations
The dataset includes the following critical fields:
- `case_id`: Unique identifier for each judgment.
- `date`: Judgment date.
- `text`: Full text of the judgment.
- `summary`: Abstract or summary of the judgment.

## Citation Structure
For any analysis published referencing this data, please cite the proper repository and Kaggle dataset as standard academic convention.
